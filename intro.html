Important Terms:

W3C
It stands for World Wide Web Consortium. It sets standards for many new web technologies that have emerged.
Attribute
They are special words used inside an opening tag to control the elements behavior.
Browser
It is a software application which is used to access information via World Wide Web.
Meta data
A set of data that describes and gives information about other data.
Responsive Design
It is a web development approach that creates dynamic changes to the appearance of a website, depending on the screen size and orientation of the device being used to view it.
